# Laboratorio Bases de Datos Avanzadas

## Autor
Andres Herrera

## Descripción
Repositorio correspondiente al laboratorio de XML, XSD y XQuery utilizando Oracle Database XE y SQL/XML.

## Contenido

- Generación de documentos XML
- Validación mediante XSD
- Consultas XPath y XQuery
- Inserciones XML en Oracle XMLTYPE
- Scripts SQL utilizados en el laboratorio

## Tecnologías utilizadas

- Oracle Database XE 21c
- Oracle SQL Developer
- SQL/XML
- XML
- XSD
- XPath
- XQuery
